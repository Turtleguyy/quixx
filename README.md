# Qwixx Score Sheet

A simple browser-based score sheet for the dice game **Qwixx**.

This is a lightweight static web app built with plain HTML, CSS, and JavaScript. No accounts, no backend, no build process — just open it and play.

## Features

- Interactive score rows for the four Qwixx colors
- Lock buttons for completed rows
- Penalty tracking
- Automatic total score display
- New game reset button
- Works in the browser with no installation required

## Live Site

[qwixx.online](https://qwixx.online)

## How to Use

Click numbers as you mark them during a game of Qwixx. Use the lock buttons when a row is locked, track penalties at the bottom, and the total score updates as you play.

To start over, click **New Game**.

## Running Locally

Clone the repo:

```bash
git clone https://github.com/Turtleguyy/quixx.git
cd quixx
```
